Git:
 1. Subir un proyecto a GitHub desde cero

git init                                                    # Inicializa el repositorio local
git remote add origin https://github.com/USUARIO/REPO.git   # Vincula con GitHub
git add .                                                   # Agrega todos los archivos
git commit -m "Primer commit"                               # Crea el primer commit
git branch -M master                                        # (Opcional) Asegura que la rama sea 'master'
git push -u origin master                                   # Sube a GitHub por primera vez




2. Subir cambios al proyecto ya existente
Cada vez que modificás algo (como HTML, JS, CSS compilado, etc.):


git add .                                      # Agrega los cambios
git commit -m "Descripción de los cambios"     # Crea un commit con mensaje
git push                                       # Sube los cambios a GitHub
